### 가상메모리

⇒ PC가 proccess에게 보여주는 가상의 메모리

가상메모리 기술이 동작하기 위해서는 2가지 문제가 있다

1. 가상 메모리를 물리 메모리에 매핑시키는 Address Translation
2. 가상 공간에 대한 관리

⇒ VM과 PM의 매핑을 위한 기술로 Paging 기법을 사용한다

### Paging

⇒ PC가 RAM에서 사용하기 위해 데이터를 저장하고 검색하는 메모리 관리 기법이다

- 물리적 메모리를 연속적으로 할당할 필요가 없다
- 연속적이지 않은 PM도 페이징을 통해 연속적으로 보이게 할 수 있다

**특징**

- VM을 일정한 크기로 분할한다
- 주소 공간은 page단위로 분리된다
- Frame (PM)과 Page (VM)의 단위는 같다

### Page table

⇒ 메모리가 할당된 Page는 Page table에 저장되어 관리된다

- Index로 해당 Frame의 시작 주소를 접근할 수 있다
- Page Table Entry는 테이블의 레코드를 의미한다
