[[Flask]]
## 라우팅 시작

```python
app.route('/')
def home():
	return 'hi'
```

⇒ `URL` 이 기본일 때 해당 라우터가 담당함

⇒ hi 를 반환함

- 예제
    
    ```python
    @app.route('/create/')
    def create(): 
        return 'create'
    
    @app.route('/read/1/')
    def read():
        return 'Read 1'
    ```
    

## 변수 라우팅 규칙

```python
@app.route('/read/<id>/')
def read(id):
    print(id)
    return 'Read' + id
```

⇒ `<>` 로 사용하고자 하는 변수 이름을 감싼다

⇒ 해당 값은 read 함수에 파라미터 값으로 들어간다, 즉 함수에서 id를 받아 사용할 수 있다

⇒ 로그는 CMD에 찍힌다
