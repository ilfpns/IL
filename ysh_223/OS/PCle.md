### Peripheral Component Interconnect Express

⇒ 시스템 내부에서 메인 보드와 주변 장치를 연결해주는 고속 직렬 데이터 통신 규격이다

⇒ 데이터가 송수신되는 초고속 버스

**왜 사용할까?**

과거 통신 방식(PCI)는 여러 장치가 하나의 버스를 이용함, 하지만 PCle는 장치마다 전용 버스를 부여하는 Point to Point 방식이다

- 속도 : Lane으로 나누어진 버스들은 빠르게 동시적으로 전송될 수 있다
- 확장성 : Lane의 폭(width)을 늘려 대역폭을 조절 할 수 있다
- 양방향 통신 : 데이터를 양쪽에서 동시에 송수신이 가능하다 (Full - DuPlex)

**구조**

- x1 : 짧은 슬롯 → 사운드 카드, WiFi 모듈
- x4 / x8 : 중간 슬롯 → 서버 네트워크, NVMe SSD
- x16 : 긴 슬롯 → GPU
