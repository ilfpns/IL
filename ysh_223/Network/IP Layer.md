## OSI 7Layers
[[Network]]
⇒ 국제표준화기구에서 만든 네트워크 통신 표준

## TCP/IP 4Layers

⇒ OSI 7Layers에 비해 간단한 TCP/IP 프로토콜을 규격화한 계층 모델

## Internet Protocol Layers

⇒ 네트워크 통신 시 데이터 전송 경로 설정, 주소 지정을 담당하는 프로토콜

**과정**

1. 데이터를 패킷이라는 단위로 분할해 전송한다
2. 각 패킷은 출발지 IP 주소와 목적지 IP주소를 가진다
3. IP 프로토콜은 이 주소를 이용해 데이터를 목적지로 전송한다
4. 그 외의 일에는 관여하지 않고, 데이터 전송 위치 결정에만 관여한다

**DataLink Layer와 다른 점이 무엇인가?**

⇒ DataLink계층도 데이터의 전송 위치를 결정하기 때문에 혼동할 수 있음

- 2계층 (DataLink) : 어느 GateWay로 갈지 정함
- 3계층 (IP Layer) : 어느 주소(IP)로 갈지 정함
- **Flow**
    1. 3계층에서 IP 확인
    2. Local IP가 아님을 인지
    3. GateWay로 보냄 (3계층 → 2계층)
    4. 2계층에서 받은 MAC주소를 확인
    5. 다음 GateWay 결정 후 전송

**GateWay란?**

⇒ 서로 다른 프로토콜을 사용하는 네트워크나 시스템 간의 통신을 가능하게 하기 위해 데이터를 변환하고 중계하는 접점
