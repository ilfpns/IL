> **ngrok**
[[Network]]
---
⇒ 로컬에서 실행 중인 서버를 인터넷을 통해 외부에서 접근할 수 있도록 해주는 터널링 도구이다

- 터널링
    
    네트워크 외부에서 내부 시스템 접근을 돕거나, 안전한 통신 채널을 만드는 또는 물리적 지하 개발을 위한 기계를 지칭
    

### 특징

- 간편한 설치와 사용
    
    ⇒ 복잡한 설정 없이 명령어 한 줄로 외부 접근이 가능하다
    
- 보안 연결 지원
    
    ⇒ TLS/SSL을 기본적으로 지원하여 안전한 데이터 전송이 된다
    
- 다양한 플랫폼 지원
    
    ⇒ 모든 운영체제에서 지원한다
    
- 무료 플랜 제공
    
    ⇒ 무료로 쓸 수 있다
    

### 사용

1. ngrok을 다운받음
2. 토큰을 발급받아 사용
