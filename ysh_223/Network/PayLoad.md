> **PayLoad**

---

- 사용에 있어서 전송되는 데이터
- 전송의 목적이 되는 데이터의 일부분으로 함께 전송되는 헤더와 메타데이터와 같은 데이터는 제외

![[Pasted image 20260116111930.png]]

- 메타데이터
    
    데이터를 설명하는 데이터
    
    EX) 택배 (데이터), 송장 (메타데이터 )
    

### 상황별 PayLoad

- 네트워크
    
    ⇒ 패킷 구조에서 헤더와 트레일러를 떼어낸 순수 데이터 영역
    
- 보안
    
    ⇒ 실행되는 악성 코드
    
- 프로그래밍 (API)
    
    ⇒ JSON의 응답 본문 자체
    

### 오버헤드

⇒ 메타데이터 및 헤더가 Payload보다 큰 것

⇒ 페이로드 대 헤더 비율을은 최적화의 핵심 기술
