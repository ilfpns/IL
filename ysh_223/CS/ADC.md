## ADC

⇒ 아날로그를 디지털로 바꾸는 것

⇒ 아날로그 값으로 들어온 데이터를 디지털 값으로 바꿈

### ADC 분해능 (세로축 : 전압의 해상도)

⇒ ADC의 입력 해상도

⇒ 얼마나 세밀하게 읽을 것인가?

- 8-bit ADC :
    - 2^8 = 256
    - 전압 변화 감지를 256단계로 쪼갠다
    - 5V / 256 = 19.5mV 단위
- 12-bit ADC :
    - 2^12 = 4096
    - 전압 변화 감지를 4096단계로 쪼갠다
    - 5V / 4096 = 1.22mV

⇒ 12bit는 8bit보다 약 16배 더 정밀하게 읽어냄

⇒ 비트 수가 높아짐에 따라 노이즈에 민감해짐

- 설계가 까다로움
- 변환 속도가 느려짐
- 모듈 가격이 비싸짐

### ADC Sampling speed (가로축 : 샘플링 속도)

⇒ 1초에 몇 번이나 전압을 읽을 것인가?

- 나이퀴스트 정리 (Nyquist Theorem
    - 원래의 아날로그 신호를 디지털로 완벽하게 복원하려면, 신호의 최고 주파수보다 최소 2배 이상 빠른 속도로 샘플링을 해야 한다는 법칙
        
    - 속도가 너무 느리면 원래 신호와 전혀 다른 엉뚱한 파열으로 잘못 읽어들임
        
        ⇒ 에일리어싱 (Aliasing)
