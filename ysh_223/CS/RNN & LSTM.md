## RNN, Recurrent Neural Network

입력과 출력을 시퀀스 단위로 처리하는 시퀀스 모델이다

⇒ 입력과 출력을 연속적인 사건, 행동, 데이터의 순서나 차례로 처리하는 모델이다

⇒ 시퀀스 : 순서, 연속성

![[Pasted image 20260329184455.png]]
다음 사진은 RNN의 기본 구조이다

⇒ ht = (ht-1 + Xt)

⇒ 내부적인 계산 (ht) 후 출력 (yt)

- 녹색 : 히든 state
- 적색 : 인풋 x
- 청색 : 아웃풋 y
- 히든 state ht는 직전 시점의 히든 state ht-1을 받아서 갱신된다
- 현재 상태의 아웃풋 yt는 ht를 전달받아 갱신되느 구조이다
- state의 활성화하수는 하이퍼볼릭탄젠트를 사용한다

![[Pasted image 20260329184508.png]]
현재 시점 t에서의 히든 state값을 ht, 히든레이어의 메모리셀은 ht를 계산하기 위해 총 두개의 가중치를 가진다

- 입력 가중치 Wx
- ht-1 가중치 Wh

---

RNN : 이전 연산에 값을 다음 값 계산에 이용한다

**문제점**

- 수학적으로 과거의 값을 계속 연산하면서 전달하다보니, 문장이 길어지면 처음 인풋이 흐려진다
    
    ⇒ 기울어진 소실 문제 발생 (오래전 입력을 기억하지 못함)
    

## LSTM, Long Short-Term Memory models

RNN의 히든 state에 cell-state를 추가한 구조이다

![[Pasted image 20260329184525.png]]
cell-state는 일종의 컨베이어 벨트 역할을 하며, state가 오래되더라고 그래디언트가 비교적 잘 전파된다

- forget gate : ft → 과거 정보를 잊기 위한 게이트로
    - ht-1과 Xt를 받아서 출력값을 0과 1 사이의 연속적인 값으로 변환한다
- Input gate : gt → 현재 정보를 기억하기 위한 게이트로
    - hadamard product(tanh(ft + 입력))
    - hadamard product : 같은 크기의 두 행렬의 각 성분을 곱하는 연산

---

Cell-State는 뭘 보관할까?

- ht (은닉층) : 이전 값을 기억, 출력용
- Ct (셀) : 문장 주제의 장기적 기억, 보관용

⇒ ht = ((ht-1 + Xt), Ct)

⇒ Ct는 이동하면서 새로 기억할 정보를 더하고, 필요 없는 데이터는 곱한다

⇒ *, + 같은 선형적 연산만 하므로 기존 데이터가 훼손되지 않음 (기울기 소실이 발생하지 않음)

---

- 기울기 소실을 막는 연산 : +, -, *
- 비선형적인 연산 : tanh, 시그모이드

⇒ RNN은 매 단계마다ht-1 + Xt 후 tanh를 하므로 기울기 소실이 발생함

⇒ LSTM은 Cell을 사용하여 선형적 연산만으로 데이터를 보존
