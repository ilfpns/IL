[[C]]
# #if #endif

⇒ 컴파일 이전 단계에서 특정 조건에 따라 코드 진행 여부를 결정함

```c
#define VERSION 2

#if VERSION == 1
    printf("Version 1 code\\n");
#elif VERSION == 2
    printf("Version 2 code\\n");
#else
    printf("Unknown version\\n");
#endif

```

다음과 같은 코드에서 VERSION2만 살아있는 코드로 취급해 컴파일된다

다른 #if문 안에 있는 코드는 실행되지 않는다

```c
#define BOARD_AITHINKER 1
#define BOARD_ESP_EYE  0

#if BOARD_AITHINKER
    #define CAM_PIN_PWDN -1
    #define CAM_PIN_RESET -1
#elif BOARD_ESP_EYE
    #define CAM_PIN_PWDN 32
    #define CAM_PIN_RESET -1
#endif

```

다음과 같은 코드는 BOARD_AITHINKER에 직접 1을 집어넣어 실행하는 구문이다

다음과 같은 코드에서 define으로 정의되는 핀이 다르기에 설정이 중요하다

- #ifdef ⇒ 매크로가 정의되었다면 참
- #ifndef ⇒ 매크로가 정의되지 않았다면 참
- #if ⇒ 일반적인 숫자 비교나 논리 연산까지 가능함

<aside>

- `#if` 조건은 전처리기 매크로 상수만 사용이 가능하다
- 코드 실행중의 `if` 문과는 다름 `if` 는 런타임에 결정되고 `#if` 는 컴파일 때 결정됨 </aside>
