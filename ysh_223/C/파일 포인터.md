### 파일 포인터 [[C]]
=> 파일 포인터는 커서와 같은 역할을 해서 파일 내부에서 원하는 위치부터 데이터를 불러올 수 있게 됩니다

```c
int main() { 
	FILE* file = fopen("Test.txt", "rt");  
	rewind(file); 
	fclose(file); 
	return 0; 
}
```
- R, W, A모드가 있음

### R (읽기)
- 만들어진 파일을 읽기 위해 처음 위치에 파일 포인터를 위치함

### W (쓰기)
- 이미 만들어진 파일 또는 새로운 파일의 가장 처음 위치한 파일의 포인터가 위치함
### A (이어 쓰기)
- 파일의 가장 마지막에 위치함
