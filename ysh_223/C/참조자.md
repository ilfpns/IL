[[C]]
```cpp
int n = 10;
int &ref = n;

ref = 20;
cout << n << endl;
```

실행 결과는 20이다 언뜻 보기에는 포인터와 같은 역할을 한다

- 상수에 대한 참조는 불가하다
- 겉보기엔 전혀 포인터로 보이지 않는다

```cpp
void func(int& ref) {
	ref = 20;
}

int main() {
	int a = 10;
	func(a);
	cout << a << endl;
}
```

이는 func함수에 a가 참조자로 들어가므로 ref에 값을 바꿨을 때 a의 값도 바뀐다
